Controls:
Left mouse button - clicking
Middle mouse button - translation
Right mouse button - rotation

Key:
Ctrl - camera movement
X - block/unblock X axis
Y - block/unblock X axis
Z - block/unblock X axis
C - clear selection
U - uniform/non-uniform scaling
Del - delete objects
Alt/A - move Z instead of XY
Shift - scaling

Creating object:
P - point
T - torus

Required libraries:
OpenGL 4.3+
GLFW 3
GLEW
and their depedencies
